# 🎮 WordFlux - Word Puzzle Game

A beautifully designed word puzzle game built with React Native Expo and Firebase.

## Features
-  Daily Challenge - 90 seconds, 3 attempts
-  Practice Mode - No timer, play freely  
-  Theme Game - Animals, Food, Science, Countries, Music, Valentine
-  Word Journey - Word of the Day with definitions
-  Guessed Words History
-  Global Leaderboard
-  Firebase Authentication

## Tech Stack
- React Native + Expo SDK 55
- Firebase Auth + Firestore
- React Navigation
- TypeScript
